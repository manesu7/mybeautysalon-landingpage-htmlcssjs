Hairdressing Webpage

    The structure of that responsive webpage is simple. Later it might be filled with more sections, features. For now it consists of following sections:
  • Navbar (Home, About us,Services,UI instructions).The logo image can be added here also.
  • Hero image
  • Section Abous us
  • Section Services/profile demonstrating haircuts/styles etc.
  • Footer with social media icons, facebook, twitter etc

Colors used

  • Colors of webpage are done in pink-black-purple-gray-white tone
  • Background color: white.
  • Text color on hero image is ivory
  • Text color in About us section is black and background color is beige.
  • Text color for Footer text is white, rgba(255,255,255,.6). Footer area's background image color varies between levels of pink and black, smooth transition between them. It has linear gradient effect due to linear-gradient(to bottom, lightpink, #000 100%)
  • Black color - Navigation menu
  • White color- navigation menu links: Home,About us,Services,UI instructions
  • Pink - for hover over menu button in navbar
